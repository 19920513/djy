<tr><td VALIGN=TOP width="626"><h2 align=center>去年第四季度珀斯市政府辖区亏损售房比例最高</h2>

<h6></h6>
<hr>
	<p>【大纪元2023年03月31日讯】<span lang="zh-CN"><span lang="zh-TW">（大纪元记者林达澳洲珀斯编译报导）</span><span lang="zh-TW">澳洲房地产数据机构</span></span>CoreLogic <span lang="zh-CN"><span lang="zh-TW">最新的房产销售盈亏报告显示，在去年第四季度，<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E7%8F%80%E6%96%AF%E6%88%BF%E4%BA%A7.md#1">珀斯房产</a>亏损销售比例</span> <span lang="zh-TW">（</span></span>13.2%<span lang="zh-CN"><span lang="zh-TW">）在首府城市中排名第二，仅次于达尔文</span> <span lang="zh-TW">（</span></span>25.5%<span lang="zh-CN"><span lang="zh-TW">）。其中珀斯独立房与单元房亏损销售差异最大</span><span lang="zh-TW">——</span></span>5.8% <span lang="zh-CN"><span lang="zh-TW">的独立房转售亏损，而</span></span>35.4%<span lang="zh-CN"><span lang="zh-TW">的单元房转售亏损。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">根据对珀斯各市政府辖区的销售情况发现，特别引人注意的是，在珀斯市政府辖区包括</span></span>Claisebrook<span lang="zh-CN"><span lang="zh-TW">、</span></span>East Perth <span lang="zh-CN"><span lang="zh-TW">和</span> </span>Crawley<span lang="zh-CN"><span lang="zh-TW">区，这些是绿树成荫的好区，超过一半（</span></span>59.4%<span lang="zh-CN"><span lang="zh-TW">）的房产都是亏本销售，卖家损失的中位数为</span></span>6.3<span lang="zh-CN"><span lang="zh-TW">万澳元；在珀斯富人区西郊</span></span>Claremont<span lang="zh-CN"><span lang="zh-TW">市政府辖区包括</span></span>Claremont <span lang="zh-CN"><span lang="zh-TW">区和</span> </span>Swanbourne<span lang="zh-CN"><span lang="zh-TW">区，有</span></span>37.5% <span lang="zh-CN"><span lang="zh-TW">的卖家亏本出售，中位亏损数为</span> </span>57,550 <span lang="zh-CN"><span lang="zh-TW">澳元；在</span></span>Belmont<span lang="zh-CN"><span lang="zh-TW">市政府辖区，包括</span> </span>Ascot <span lang="zh-CN"><span lang="zh-TW">和</span> </span>Rivervale <span lang="zh-CN"><span lang="zh-TW">郊区，有</span> </span>29.2% <span lang="zh-CN"><span lang="zh-TW">的房地产销售亏损，中位数达到</span> </span>45,000 <span lang="zh-CN"><span lang="zh-TW">澳元。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">其它大量房产销售出现亏损的市政府辖区包括</span> </span>Vincent <span lang="zh-CN"><span lang="zh-TW">（</span></span>28.4%<span lang="zh-CN"><span lang="zh-TW">）、</span></span>Subiaco <span lang="zh-CN"><span lang="zh-TW">（</span></span>28.3%<span lang="zh-CN"><span lang="zh-TW">）</span> <span lang="zh-TW">和</span> </span>Victoria Park <span lang="zh-CN"><span lang="zh-TW">（</span></span>20.4%<span lang="zh-CN"><span lang="zh-TW">）。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">珀斯去年第四季度的亏损销售额全澳最高，在全国亏损销售额亏损中占比近</span> </span>20%<span lang="zh-CN"><span lang="zh-TW">。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">然而，在全国范围内，</span></span>93.1% <span lang="zh-CN"><span lang="zh-TW">的房产转售同时实现账面利润，但住宅房地产的盈利率在本季度略有恶化，亏损从</span> </span>6.6% <span lang="zh-CN"><span lang="zh-TW">上升至</span> </span>6.9%<span lang="zh-CN"><span lang="zh-TW">。</span><span lang="zh-TW">?</span></span></p>
<p>Core Logic <span lang="zh-CN"><span lang="zh-TW">澳洲研究部负责人伊丽莎</span></span>·<span lang="zh-CN"><span lang="zh-TW">欧文</span> <span lang="zh-TW">（</span></span>Eliza Owen<span lang="zh-CN"><span lang="zh-TW">）表示，虽然珀斯亏损销售的比例仍然很高，但随着时间的推移，这一比例一直呈下降趋势。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">“从</span></span>2014 <span lang="zh-CN"><span lang="zh-TW">年至</span> </span>2019 <span lang="zh-CN"><span lang="zh-TW">年房价持续低迷之后，整个珀斯的十年平均亏损销售率相对较高，为</span> </span>21%<span lang="zh-CN"><span lang="zh-TW">”，她说，“即使单元房销售亏损率的高位也远低于</span> </span>2020 <span lang="zh-CN"><span lang="zh-TW">年</span> </span>6 <span lang="zh-CN"><span lang="zh-TW">月季度</span> </span>59.1% <span lang="zh-CN"><span lang="zh-TW">的峰值。”</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">欧文表示，随着大珀斯最近的资本增长趋势，整个单元房部门的盈利能力恶化，其中公寓价格在</span> <span lang="zh-TW">去年第四季度下降了</span> </span>1.2%<span lang="zh-CN"><span lang="zh-TW">，而同期独立屋价格略有上升。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">她说：“此外，在大流行带来的创纪录低利率背景下，珀斯的单元房价值从未像独立房房价那样创下历史新高。这增加了公寓部门长期持有的可能性，以应对转售亏损。”</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">珀斯房地产专家弗莱斯肯斯</span> <span lang="zh-TW">（</span></span>Trent Fleskens<span lang="zh-CN"><span lang="zh-TW">）</span> <span lang="zh-TW">表示，独立房比公寓更有可能获利。</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">“在西澳，平均持有房产的时间约为</span> </span>11 <span lang="zh-CN"><span lang="zh-TW">年，我们在这里看到的是，尤其是在这些郊区，许多房主多年来一直以负资产形态持其房产</span><span lang="zh-TW">——</span><span lang="zh-TW">通常是投资公寓和单元房。并认为现在是出手再次接近盈亏平衡点的合适时机。”他说。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">“这反映了这样一个事实，即在市中心的单元房和公寓中，我们现在才回到十年前的价位。“将这与此间价值翻了一番的东海岸相比，你就会明白，这些销售更多是出自筋疲力尽的房东脱手，而不是无力负担抵押贷款的自住房主。”</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">弗莱斯肯斯表示，整个珀斯的房地产价格在</span> <span lang="zh-TW">第四季度上涨，而且他仍然听说每周都有新的郊区记录被打破。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">“储备银行是否真的会再提高现金利率两次是值得商榷的”，他说，“鉴于近期全球金融市场的不稳定，市场现在开始比以往更加确定未来</span> </span>12 <span lang="zh-CN"><span lang="zh-TW">个月的利率下降。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">“尽管如此，鉴于持续创纪录的每周高需求数据和创纪录的低供应水平，即使加息，也有人认为珀斯的价格增长将会放缓，但很难看到珀斯建材全面价格下跌。”</span><span lang="zh-TW">?</span></span></p>
<p>Property Powerhouse <span lang="zh-CN"><span lang="zh-TW">首席执行官戴维斯</span><span lang="zh-TW"> （</span></span>Garth Davis<span lang="zh-CN"><span lang="zh-TW">） </span><span lang="zh-TW">表示，过去几年，珀斯没有经历过悉尼、墨尔本、布里斯班、霍巴特和阿德莱德那样的价格上涨。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">“但我们的时代即将到来”，他说，“一旦利率在今年</span> </span>6 <span lang="zh-CN"><span lang="zh-TW">月左右稳定下来，这将是转折点，我预计在未来三到四年内，珀斯房地产市场将出现良好的价格增长。成千上万的购房者正坐等利率达到上限，然后将出售自己的房子并寻求升级。</span>”</span></p>
