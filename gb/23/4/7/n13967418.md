<tr><td VALIGN=TOP width="626"><h2 align=center>西澳疫苗安全监测 2021 年度报告延期半年公布</h2>

<h6></h6>
<hr>
	<p>【大纪元2023年03月31日讯】<span lang="zh-CN"><span lang="zh-TW">（大纪元记者林达澳洲珀斯编译报导）</span><span lang="zh-TW">西澳于</span> </span>2021 <span lang="zh-CN"><span lang="zh-TW">年</span> <span lang="zh-TW">中共病毒（</span></span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">）<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E7%96%AB%E8%8B%97%E6%8E%A5%E7%A7%8D.md#1">疫苗接种</a>纳入西澳的免疫计划后，导致与</span> </span>2020 <span lang="zh-CN"><span lang="zh-TW">年相比，疫苗不良反应报告增加了</span> </span>10,500 <span lang="zh-CN"><span lang="zh-TW">份。</span></span>2021 <span lang="zh-CN"><span lang="zh-TW">年，西澳共接种了</span> </span>575 <span lang="zh-CN"><span lang="zh-TW">万剂疫苗，高于</span> </span>2020 <span lang="zh-CN"><span lang="zh-TW">年的</span> </span>207 <span lang="zh-CN"><span lang="zh-TW">万剂。其中，其中</span></span>394 <span lang="zh-CN"><span lang="zh-TW">万是单独剂量的中共病毒（</span></span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">）疫苗。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">延迟六个月后，<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E8%A5%BF%E6%BE%B3%E7%96%AB%E8%8B%97.md#1">西澳疫苗</a>安全监测</span> </span>2021 <span lang="zh-CN"><span lang="zh-TW">年度报告于上月发布，报告显示接种疫苗后的不良反应范围从轻微的头痛到过敏反应，甚至有一个死亡案例。然而，有专家表示，报告反映了<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E7%96%AB%E8%8B%97%E6%8E%A5%E7%A7%8D.md#1">疫苗接种</a>的规模、改进了围绕<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E7%96%AB%E8%8B%97%E5%AE%89%E5%85%A8.md#1">疫苗安全</a>的公众教育、以及改进了报告反应的机制。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW"><ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E5%85%8D%E7%96%AB%E6%8E%A5%E7%A7%8D.md#1">免疫接种</a>后发生的</span> </span>10,726 <span lang="zh-CN"><span lang="zh-TW">起不良反应事件（称为</span> </span>AEFI<span lang="zh-CN"><span lang="zh-TW">）中，有</span> </span>10,428 <span lang="zh-CN"><span lang="zh-TW">起是在接种</span> </span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">疫苗后记录的。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">每接种</span> </span>10<span lang="zh-CN"><span lang="zh-TW">万剂</span> </span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">疫苗，就会记录</span> </span>264 <span lang="zh-CN"><span lang="zh-TW">次</span> </span>AEFI<span lang="zh-CN"><span lang="zh-TW">；比率为</span></span>0.26%<span lang="zh-CN"><span lang="zh-TW">。比率因</span> </span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">疫苗的类型而异。在年轻人出现凝血报告后，阿斯利康疫苗（</span></span>AstraZeneca<span lang="zh-CN"><span lang="zh-TW">）几乎被淘汰，记录的</span> </span>AEFI <span lang="zh-CN"><span lang="zh-TW">率为每</span> </span>10<span lang="zh-CN"><span lang="zh-TW">万人</span> </span>306 <span lang="zh-CN"><span lang="zh-TW">例，而辉瑞疫苗（</span></span>Pfizer<span lang="zh-CN"><span lang="zh-TW">）为</span> </span>244 <span lang="zh-CN"><span lang="zh-TW">例。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">其余免疫计划的</span> </span>AEFI <span lang="zh-CN"><span lang="zh-TW">发生率为每</span> </span>10 <span lang="zh-CN"><span lang="zh-TW">万人</span> </span>11.1 <span lang="zh-CN"><span lang="zh-TW">例，该计划在</span> </span>2021 <span lang="zh-CN"><span lang="zh-TW">年共接种了约</span> </span>180 <span lang="zh-CN"><span lang="zh-TW">万剂疫苗。</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">报告指出，疫苗接种后的不良事件可能很轻微，例如手臂酸痛，但并不是由接种疫苗引起的决定性证据。在报告的</span> </span>AEFI <span lang="zh-CN"><span lang="zh-TW">中，头痛最为常见，每</span> </span>10<span lang="zh-CN"><span lang="zh-TW">万剂有</span> </span>70 <span lang="zh-CN"><span lang="zh-TW">份报告，其次是嗜睡，</span></span>60.4 <span lang="zh-CN"><span lang="zh-TW">份报告，肌肉痛</span></span>50.7<span lang="zh-CN"><span lang="zh-TW">份。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">报告的最严重反应中，每</span> </span>10<span lang="zh-CN"><span lang="zh-TW">万剂有</span> </span>1 <span lang="zh-CN"><span lang="zh-TW">例过敏反应，而血栓形成的记录率为每</span></span>10<span lang="zh-CN"><span lang="zh-TW">万剂</span> </span>2.1 <span lang="zh-CN"><span lang="zh-TW">例，心肌炎和心包炎有</span></span>0.2<span lang="zh-CN"><span lang="zh-TW">例。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">在整个</span> </span>2021 <span lang="zh-CN"><span lang="zh-TW">年疫苗推广过程中，只发现一例死亡与疫苗接种有因果关系。自大流行开始以来，西澳记录了</span> </span>985 <span lang="zh-CN"><span lang="zh-TW">例与</span> </span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">相关的死亡。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">报告称，</span></span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">疫苗接种计划在</span> </span>2021 <span lang="zh-CN"><span lang="zh-TW">年引起了相当大的公众兴趣和媒体关注，这几乎肯定会提高公众对<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E7%96%AB%E8%8B%97%E5%AE%89%E5%85%A8.md#1">疫苗安全</a>的认知。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">“</span></span>2021 <span lang="zh-CN"><span lang="zh-TW">年</span> </span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">疫苗接种后的大量报告反映了接种率的提高，以及公众和医保提供者对疫苗安全性监测的高度参与。”报告说。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">在</span> </span>COVID-19 <span lang="zh-CN"><span lang="zh-TW">疫苗接种推广期间，报告</span> </span>AEFI <span lang="zh-CN"><span lang="zh-TW">的机制得到了加强。其中包括要求接种后等待</span> </span>15 <span lang="zh-CN"><span lang="zh-TW">分钟再报告，以及来自</span> <span lang="zh-TW">西澳卫生厅（</span></span>WA Health<span lang="zh-CN"><span lang="zh-TW">）的后续短信。卫生厅还建立了一个专门的疫苗安全团队，并改进了对卫生系统数据的使用。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">卫生厅厅长桑德森</span> <span lang="zh-TW">（</span></span>Amber-Jade Sanderson<span lang="zh-CN"><span lang="zh-TW">）的发言人表示，与大流行以来接种的数百万剂疫苗相比，接种后不良反应数量不足为虑。她说，还应考虑到最近加强了疫苗监测和报告这一事实。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">科廷大学学院公共卫生研究员贾亚</span></span>·<span lang="zh-CN"><span lang="zh-TW">丹塔斯</span> <span lang="zh-TW">（</span></span>Jaya Dantas<span lang="zh-CN"><span lang="zh-TW">）教授对较高的</span> </span>AEFI <span lang="zh-CN"><span lang="zh-TW">发生率并不感到惊讶，考虑到疫苗推出规模和疫苗周转时间。“我认为，不需要担心，但需要继续监测，去支持那些有不良反应的人。”她说。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">上议院自由党议员戈兰</span> <span lang="zh-TW">（</span></span>Nick Goiran<span lang="zh-CN"><span lang="zh-TW">）表示，与前几年相比，</span></span>AEFI <span lang="zh-CN"><span lang="zh-TW">的数量不成比例地增加，即使考虑到接种数量，也引发了对需要进一步调查的质疑。</span><span lang="zh-TW">?</span></span></p>
<p><span lang="zh-CN"><span lang="zh-TW">西澳卫生厅的一位女发言人表示，已对报告的不良事件进行严格调查。“西澳不良事件数据将发送给治疗用品管理局</span> </span>(TGA)<span lang="zh-CN"><span lang="zh-TW">，以促进国家疫苗安全监测并提高安全性。”她说。</span><span lang="zh-TW">?</span></span></p>
