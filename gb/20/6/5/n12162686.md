<tr><td VALIGN=TOP width="626"><h2 align=center>滑雪季 这些城市间航班将恢复到55%</h2>
<h6></h6>
<hr>
<p>【大纪元2020年06月05日讯】( 大纪元记者章丰<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>编译报导 )新西兰<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E8%88%AA%E7%A9%BA%E5%85%AC%E5%8F%B8.md#1">航空公司</a>宣布，计划在 7 月和 8 月期间，将<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E5%9B%BD%E5%86%85%E8%88%AA%E7%8F%AD.md#1">国内航班</a><ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E8%BF%90%E8%A1%8C.md#1">运行</a>时间维持在疫情爆发前的 55 % 左右。</p>
<p>自国家进入 2 级警报以来，<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E8%88%AA%E7%A9%BA%E5%85%AC%E5%8F%B8.md#1">航空公司</a>已经飞往国内大多数港口，尽管频率有所降低。</p>
<p><ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E6%96%B0%E8%A5%BF%E5%85%B0.md#1">新西兰</a>航空公司网络总经理 斯科特 ? 卡尔 ( Scott Carr ) 表示 : “ 最近休闲旅行者的需求鼓舞了我们，我们也预计商务旅行的需求将继续增长。”</p>
<p>新计划还将包括航空公司的因弗卡吉尔 &#8211; 奥克兰 A 320 喷气式飞机往返航班，航班每周<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E8%BF%90%E8%A1%8C.md#1">运行</a>四次。</p>
<p>公司还将于7月初重新推出其因弗卡吉尔 ( Invercargill ) &#8211; 惠灵顿 ( Wellington ) , 基督城 ( Christchurch ) &#8211; 罗托鲁瓦 ( Rotorua ) 和基督城 ( Christchurch ) &#8211; 新普利茅斯 ( New Plymouth ) 航线。</p>
<p>蒂马鲁 ( Timaru ) &#8211; 惠灵顿 ( Wellington )和塔坡 (Taupo) &#8211; 奥克兰 ( Auckland ) 航线将于周一<ahref="https://github.com/19920513/djy/blob/master/gb/tag/%E6%81%A2%E5%A4%8D.md#1">恢复</a>。</p>
